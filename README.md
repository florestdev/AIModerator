# AIModerator
ИИ-модератор для чата вашего minecraft-сервера.
# Информация
Версия: 1.21+
Java: 21+
Вес: 70~ МБ
# Как пользоваться?
Получите API ключ с этого сайта: https://app.fireworks.ai/login?redirectURI=%2Fsettings%2Fusers%2Fapi-keys (в России работа сервиса ограничена).
Вставьте ключ в параметр `api_key` в config.yml.
Можно настроить наказание для каждого отдельного правила Вашего сервера.
Пример:
```
rules:

  rule1:
    name: "Не вести себя токсично"
    description: "Запрещены оскорбления, маты, унижения в чате"
    threshold: 0.6

    action: "command"
    action_message: "Вы нарушили правило: %rule%"
    command: "mute %player% 30m Нарушение правил чата"

  rule2:
    name: "Реклама только официальных ресурсов"
    description: "Запрещена реклама сторонних серверов и ресурсов в чате"
    threshold: 0.7

    action: "command"
    action_message: "Вы нарушили правило: %rule%"
    command: "mute %player% 30m Нарушение правил чата"

  rule3:
    name: "Ненависть к группам"
    description: "Запрещены оскорбления или проявления ненависти к любым группам в чате"
    threshold: 0.6

    action: "command"
    action_message: "Вы нарушили правило: %rule%"
    command: "mute %player% 30m Нарушение правил чата"

  rule4:
    name: "Деанонимизация"
    description: "Запрещено раскрывать личные данные игроков и админов в чате"
    threshold: 0.6

    action: "command"
    action_message: "Вы нарушили правило: %rule%"
    command: "mute %player% 30m Нарушение правил чата"

```
# Что нужно?
Подключение к Интернету, НЕ-российский IP адрес, 1.21+, Java 21+
# Ложные срабатывания
Четко указывайте, что вы хотите от нейросети.
